Receipt Reader

Scan receipts from your phone and extract line items.

Angular PWA uploads the photo, .NET API orchestrates the pipeline, Go OCR performs the first pass, and Gemini optionally refines uncertain fields.

Pipeline

  1. Capture or upload an image
  2. OCR and heuristic parsing
  3. Optional Gemini refinement
  4. Archive the receipt with confidence markers
